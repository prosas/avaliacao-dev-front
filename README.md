Avaliação dev front
-------------------

Abaixo segue o teste para a vaga. Se tiver alguma dúvida e não conseguir fazer alguma parte, pode pular e fazer o 
máximo que conseguir. O resultado deverá ser publicado em qualquer repositório público git (github, bitbucket, 
gitlab, etc...) com as instruções necessárias para rodar o projeto. 

Especificamos quais tecnologias você deve usar na implementação, mas fique a vontade para usar qualquer outra que 
ache adequada para complementar/melhorar a solução.

**As tecnologias requeridas são:**

- React
- Redux

Teste
-----

A pessoa deve implementar um construtor de formulário - como o google forms - onde seja possível 
adicionar as perguntas e em outra aba respondê-las. Segue os requisitos:

- O formulário deve suportar um número ilimitado de perguntas
- O formulário deve suportar dois tipos de perguntas:
   - Pergunta aberta que deve aceitar como resposta um texto.
   - Pergunta múltipla escolha que deve permitir adicionar uma quantidade ilimitada de opções e deve aceitar somente 1 como resposta.
- Deve ser possível responder as perguntas definidas no formulário em uma outra aba de visualização.

## O que vamos avaliar

- Seu conhecimento do framework.
- Organização do projeto.
- Como gerencia o estado e a estrutura de dados da aplicação.
- Testes.
- README explicativo.
